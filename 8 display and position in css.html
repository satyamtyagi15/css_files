Display in css

display is nothing but the content that we see on our webpages or screen

    There are 3 types of displays in css :
  1. block scope (default)
  2. inline scope
  3. inline - block

now what's the difference b/w block scope display and inline display
  1. block scope : it always take the entire width of the screen for example if i'm displaying 2 headings then it will display the 2nd heading in the next line as it's default display which always take entire width of the screen
  2. inline scope : it always take the space that is required for example if i'm displaying the two different heading using inline display in css then it will reflect or display both the heading in the same line . it will also not take the addon height or width if provided.
  3. Inline-block : it's made by using both the displays or it's the combination of both . now it will work as a inline display untill we provide it an additional height and width.
