spacing in box model

there are two types of spacing in box model that are
1. internal spacing known as padding
2. external spacing known as margin

Internal spacing or padding in box model

for padding we need to use padding style in selector in css then we need to specify the size of spacing for each side by default it's top-right-bottom-left if you're mentioning padding 10px 20px 15px 16px then it will be exexuted according to default setting . but you can set it according to you for example padding - top : 5px ;
