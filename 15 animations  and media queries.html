animations and media queries

what is media queries so it's used to create a website in such a way that it's able to fit in any of the screensize of any of the device either it's tab,laptop,phone etc

media queries is mainly used in order to build a resposive website based on different screen size in different devices for that we can have some custom styles triggered accordingly. based on our screen size we can have different amount of styles for our device.
